# Functional Requirements

## User Management

### FR-001
Users can register for an account.

### FR-002
Users can log in using email and password.

### FR-003
Users can edit their profile.

### FR-004
Users can upload an avatar.

---

## Game Catalog

### FR-005
Users can browse available games.

### FR-006
Users can search for games.

### FR-007
Users can filter games by:
- Genre
- Platform
- Price
- Rating

### FR-008
Users can view detailed game information.

---

## Reviews

### FR-009
Authenticated users can leave reviews.

### FR-010
Users can edit their own reviews.

### FR-011
Users can delete their own reviews.

### FR-012
Users can like helpful reviews.

---

## Purchases

### FR-013
Users can purchase games.

### FR-014
Purchased games appear in the user's library.

### FR-015
Users can view their purchase history.

---

## Library

### FR-016
Users can view all owned games.

### FR-017
Users can mark owned games as favorites.

---

## Wishlist

### FR-018
Users can add games to their wishlist.

### FR-019
Users can remove games from their wishlist.

### FR-020
Users receive a notification when a wishlisted game goes on sale.

---

## Friends & Social

### FR-021
Users can send friend requests.

### FR-022
Users can accept or reject friend requests.

### FR-023
Users can remove friends.

### FR-024
Users can view another user's public profile.

---

## Notifications

### FR-025
Users receive notifications when:
- They receive a friend request.
- A purchase is completed.
- One of their reviews receives a like.
- A followed game receives an update.
- A game on their wishlist goes on sale.
- An achievement is unlocked.

---

## Achievements

### FR-026
Users can unlock achievements by completing specific milestones.

Example achievements include:
- First Purchase
- First Review
- Five Reviews Written
- Own 10 Games
- Own 25 Games
- Own 50 Games
- Wishlist Creator (add 10 games to wishlist)
- Social Gamer (add 5 friends)

### FR-027
Users can view all unlocked achievements.

### FR-028
Users can track progress toward locked achievements.

---

## Administration

### FR-029
Administrators can manage games.

### FR-030
Administrators can manage users.

### FR-031
Administrators can create and manage discounts.

### FR-032
Administrators can moderate reported reviews.
